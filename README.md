AI Document Assistant

Model Information
Model Name: Gemini 3 Flash Preview
Source: Google via the Google Gemini API

Rationale for Model Selection
I selected Gemini 3 Flash Preview because it is optimized for fast responses and works well with document-based tasks. The model supports multimodal inputs and integrates easily with Python applications using the Google GenAI SDK. It is also suitable for interactive applications like chat assistants where users ask questions about uploaded documents. Additionally, Gemini provides strong natural language understanding, which helps generate accurate answers based on the content of the uploaded file.

Responsible AI Reflection:
When building AI applications, it is important to consider responsible AI practices such as transparency, accuracy, and user privacy. In this project, the AI assistant is designed to answer questions only using the uploaded document rather than generating information from external sources. This helps reduce hallucinations and encourages more reliable responses. Additionally, users should avoid uploading sensitive or private documents because the files are processed by an external AI service. Developers also have a responsibility to clearly communicate the limitations of AI systems so users understand that responses may not always be perfectly accurate. Responsible AI use means designing systems that prioritize safety, reliability, and ethical handling of user data while still providing useful functionality.
